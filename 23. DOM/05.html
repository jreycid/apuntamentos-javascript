Eventos

Os eventos son cousas que suceden no sistema que se está a programar.
O sistema encárgase de producir unha sinal que conleva á execución automática de certo código cando se produce un evento.

Os eventos engádense con addEventListener:

            elemento_que_escoita.addEventListener('evento', funcion_a_executar);

e elimínanse con removeEventListener:

            elemento_que_escoita.removeEventListener('evento', funcion_a_eliminar);

Tipos de eventos

  • Eventos do rato
  • Eventos do teclado
  • Eventos táctiles
  • Eventos de formulario
  • Eventos de ventana
  • Eventos propios

Táboa de Eventos

Evento Descrición Elementos para os que está definido
onblur Un elemento perde o foco <button>, <input>, <label>, <select>, <textarea>, <body>
onchange Un elemento foi modificado <input>, <select>, <textarea>
onclick Pulsar e soltar o rato Todos os elementos
ondblclick Pulsar dúas veces seguidas co rato Todos os elementos
onfocus Un elemento obtén o foco <button>, <input>, <label>, <select>, <textarea>, <body>
onkeydown Pulsar unha tecla e non soltarla Elementos de formulario e <body>
onkeypress Pulsar unha tecla Elementos de formulario e <body>
onkeyup Soltar unha tecla pulsada Elementos de formulario e <body>
onload Páxina cargada completamente <body>
onmousedown Pulsar un botón do rato e non soltalo Todos os elementos
onmousemove Mover o rato Todos os elementos
onmouseout El rato "sale" do elemento Todos os elementos
onmouseover O rato "entra" no elemento Todos os elementos
onmouseup Soltar o botón do rato Todos os elementos
onreset Inicializar o formulario <form>
onresize Modificar o tamaño da ventana <body>
onselect Seleccionar un texto <input>, <textarea>
onsubmit Enviar o formulario <form>
onunload Abandónase a páxina, por exemplo o cerrar o navegador <body>
